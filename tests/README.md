# Tests

This folder records the deterministic behaviors that should remain stable as
the harness evolves. The test focus is not investment performance; it is whether
entity resolution, identifiers, schemas, citations, source freshness, and answer
shape remain under engineering control.

Test deterministic behavior first:

- group resolution
- ticker/corp-code mapping
- schema validation
- date handling
- number/unit conversion
- citation checks
- stale source warnings
- response shape validation
