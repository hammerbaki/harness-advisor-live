## 3. 판단 — 벤치마킹 vs 스코프 아웃

**벤치마킹할 것 (저비용·연구 가치 있음)**
- 대화형 멀티턴, 인용 표시 UX, 리서치 메모리/노트, 공유 문서화 워크플로.

**의도적으로 쫓지 않을 것 (비용 해자·연구 무관)**
- FnGuide/KOSCOM 등 유료 기관 데이터, 미국 시장 광범위 커버리지, 실서비스 규모.

**우리만의 방어 축 (= 논문 신규성)**
- 코드로 강제된 비추천 + 출처 계약 + 검증/재현 자산. 상업 제품이 검증 가능하게
  보여주지 못하는 것.

## 4. 미확정 항목 (직접 확인 필요)
- THR의 출처 계약이 실제 검증되는지, 규제 등록(유사투자자문업) 여부 — 약관/개인정보 페이지가
  클라이언트 렌더링이라 자동 확인 불가.
- GEN/KR-FIN 열은 대표 서비스 기준의 일반화 — 개별 제품 확정 시 갱신.
